Environment variables

  • System variables

  • User variables

  • Path variable

Java SDK

    oracle.com

  1. Downloads -> Java SE -> version 8xxx

  2. Install

  3. Add the JDK path to system PATH environment variable

  4. Test the installation with command: java -version

Apache Maven

    Apache Maven

  1. Download the latest binary zip

  2. Unzip to a directory of you choice

  3. Add the bin dir of the installation to PATH

  4. Create and new env variable called JAVA_HOME pointing to the JDK path

  5. Test the Maven installation with command: mvn--version

IntelliJ IDEA

    jetbrains.com

  1. Download IntelliJ IDEA Community Edition

  2. Install

GIT

    git-scm.com

  1. Create a new user environment variable called HOME, pointing to your user’s home directory (C:\Users\username)

  2. Add the GIT install path to PATH environment variable

  3. Test the GIT installation with command: git

  4. Configure GIT with commands:

    5. git config--global user.name “your user”
    git config--global user.email“your email”
